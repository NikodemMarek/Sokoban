saveCustomGame(gameName, levelName, worker, boxes, movesMade, movesUndone)

Funkcja konwertuje dane o grze w trybie 3 do postaci tekstowej i zapisuje je do localStorage. Zapisane dane dostają klucz rozpoczynający się od tekstu zdefiniowanego w CUSTOM_SAVE_PREFIX, a kończący się podaną nazwą zapisu.
Parameters:
Name Type Description
gameName string Nazwa zapisu gry, jeśli zapis o tej nazwie istnieje, zostanie nadpisany

worker module:worker#Worker Worker
boxes module:boxes#Boxes Boxes
movesMade number Liczba wykonanych ruchów
